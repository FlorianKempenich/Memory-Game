# Memory Game

![](MemoryGame.gif)

Simple Memory Game to practice FE dev.

## Quick start
### Live version
http://floriankempenich.com:7000

### From source
1. `$ git clone git@github.com:FlorianKempenich/Memory-Game.git`
1. `$ cd Memory-Game`
1. `$ npm install -g yarn`
1. `$ yarn install`
1. `$ yarn start`
